👤 Профиль пользователя
📱 1. Установите приложение
Выберите ваше устройство и скачайте подходящий клиент:
📱 HApp (iOS) 🤖 HApp (Android) 🚀 v2rayTun (Android) 📦 v2rayTun (Windows/macOS) ⚡ Shadowrocket (iOS)
📋 2. Добавьте подписку
Нажмите на кнопку для вашего приложения — подписка добавится автоматически:
📱 HApp (iOS/Android) 🚀 v2rayTun ⚡ Shadowrocket 💻 Clash

📖 Или вручную:

1. Скопируйте ссылку подписки (она отображается вверху страницы).

2. Откройте приложение → Добавить подписку → Вставить ссылку → Сохранить

3. Обновите подписку (потяните вниз или нажмите ⋮ → Обновить)

• В v2rayTun: Группы → Добавить группу с удалённым файлом → вставьте ссылку

📊 3. Настройка Ping (проверка задержки)
TCP Ping — лучший вариант для реальной оценки скорости.

⚙️ Как настроить (один раз):

• В HApp: Нажмите на кнопку "настройки"/⚙️ → нажмите кнопку "Пинг" → выберите TCP

• В v2rayTun: Нажмите на сервер → Тест задержки → выберите TCP Ping

• В Shadowrocket: Нажмите на сервер → Test Latency → TCP

💡 Совет: Включите "Ping on Launch" в настройках, чтобы приложение автоматически проверяло все сервера при запуске.

🛡️ 4. Максимальная маршрутизация (Всё через прокси)
Готовый профиль маршрутизации, который направляет весь трафик через прокси для максимальной анонимности.
🌍 Импорт MAX маршрутизации (HApp)

🔥 Что делает этот профиль: весь трафик идёт через прокси (кроме локальных сетей). Максимальная скрытность.

🛠️ Альтернатива для v2rayTun: в настройках маршрутизации выберите "Global Proxy".

⚠️ Ошибка 401 / Timeout / Превышено время ожидания

1. Скопируйте следующий JSON-скрипт (нажмите на блок, чтобы скопировать):

{
  "remarks": "UPLOAD SUB",
  "inbounds": [{"port":10808,"protocol":"socks","settings":{"udp":true}}],
  "outbounds": [{
    "protocol":"vless",
    "settings":{"vnext":[{"address":"test.iron-coral.test-cdn-kkk.com","port":443,"users":[{"id":"5ae46adf-322d-4e8a-8c28-3f71ce56cd9d","flow":"xtls-rprx-vision","encryption":"none"}]}]},
    "streamSettings":{"network":"tcp","security":"reality","realitySettings":{"serverName":"tradingview.com","fingerprint":"chrome","publicKey":"-tePObR3oZwGAUOb5kqTYkNWl6rtUKl0RFuzuu06wgw","shortId":"50"}}
  }]
}

2. Зайдите в приложение HApp / v2rayTun:

HApp: нажмите ➕ → «Вставить JSON» / «Вставить из буфера обмена».

v2rayTun: нажмите ➕ → ручной ввод → вставьте скопированный JSON → добавить.

3. Найдите в приложении сервер с названием «UPLOAD SUB» и запустите его.

4. Обновление подписки: если серверы пропали, нажмите справа от названия подписки кнопку 🔄 (обновить). Подписка успешно обновится.
