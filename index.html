📱 1. Установите приложение
Выберите ваше устройство и скачайте подходящий клиент:
📱 HApp (iOS) 🤖 HApp (Android) 📦 v2rayNG (Android) 🚀 Shadowrocket (iOS) 💻 v2rayN (Windows)
📋 2. Добавьте подписку
Нажмите на кнопку для вашего приложения — подписка добавится автоматически:
📱 HApp (iOS/Android) 📦 v2rayNG 🚀 Shadowrocket 💻 Clash

📖 Или вручную:

1. Скопируйте ссылку: https://raw.githubusercontent.com/citer1852-lab/SHUR-SUB/main/sub.txt

2. Откройте приложение → Добавить подписку → Вставить ссылку → Сохранить

3. Обновите подписку (потяните вниз или нажмите ⋮ → Обновить)

📊 3. Настройка Ping (проверка задержки)
TCP Ping — лучший вариант для реальной оценки скорости, он работает через VPN-туннель и показывает реальную задержку до сервера [citation:3].

⚙️ Как настроить (один раз):

• В HApp: Нажмите на кнопку "настройки"/⚙️ → нажмите кнопку "Пинг" → выберите TCP[citation:3]

• В v2rayNG: Нажмите на иконку ⚡ рядом с сервером → выберите TCP Ping

• В Shadowrocket: Нажмите на сервер → Test Latency → TCP

💡 Совет: Включите "Ping on Launch" в настройках, чтобы приложение автоматически проверяло все сервера при запуске [citation:3].

🧭 4. Импорт маршрутизации (для HApp)
Маршрутизация определяет, какие сайты идут через VPN, а какие — напрямую. Готовый профиль с настройками "всё через прокси".
🌍 Импорт маршрутизации (HApp)

📌 Что делает этот профиль: Весь трафик направляется через прокси. Российские сайты и приватные IP идут напрямую (для стабильной работы).

🛠️ Как импортировать вручную: HApp → Настройки → Маршрутизация → ⋮ → Импорт → Вставить ссылку или код

• В v2rayNG/Shadowrocket: Маршрутизация настраивается отдельно в правилах (Routing Rules).
