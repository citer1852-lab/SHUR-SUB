📊 3. Настройка Ping (проверка задержки)
TCP Ping — лучший вариант для реальной оценки скорости, он работает через VPN-туннель и показывает реальную задержку до сервера.

⚙️ Как настроить (один раз):

• В HApp: Нажмите на кнопку "настройки"/⚙️ → нажмите кнопку "Пинг" → выберите TCP

• В v2rayTun: Нажмите на сервер → Тест задержки → выберите TCP Ping

• В Shadowrocket: Нажмите на сервер → Test Latency → TCP

💡 Совет: Включите "Ping on Launch" в настройках, чтобы приложение автоматически проверяло все сервера при запуске.

🛡️ 4. Максимальная маршрутизация (Всё через прокси)
Готовый профиль маршрутизации, который направляет весь трафик через прокси для максимальной анонимности и обхода блокировок.
🌍 Импорт MAX маршрутизации (HApp)

🔥 Что делает этот профиль:

• Всё через прокси — абсолютно весь трафик (браузер, приложения, игры, обновления) идёт через VPN-сервер

• Исключения: только локальные сети (192.168.x.x, 10.x.x.x) и localhost — чтобы не сломать доступ к роутеру и локальным устройствам

• Для опытных пользователей: полный контроль над трафиком, максимальная защита от DPI и блокировок

🛠️ Альтернатива для v2rayTun:

В настройках маршрутизации выберите "Global Proxy" (Глобальный прокси) или "Proxy All". Это включит режим "весь трафик через прокси".

⚠️ Внимание: При использовании этого профиля весь ваш трафик идёт через прокси. Это может незначительно увеличить задержку, но обеспечивает максимальную скрытность.
