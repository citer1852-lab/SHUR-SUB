📋 Инструкция

  • 1. Введите имя файла (латиница, без пробелов, без .json)
  • 2. Нажмите «Проверить» — скрипт покажет, существует ли уже такой файл
  • 3. Вставьте JSON конфигурацию сервера
  • 4. Нажмите «Сохранить на GitHub» — файл будет добавлен в папку sources
  • 5. После добавления обновите кэш Cloudflare (кнопка в мониторинге)
